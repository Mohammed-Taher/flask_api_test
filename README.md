# Flask API

## Installation

1. Create a virtual environment using:

```
python -m venv venv
```

or

```
python3 -m venv venv
```

1. Install the required packages using:

```
pip install -r requirements.txt
```

1. Export the following environment variable:

```
export FLASK_ENV=develpment
```

1. Run the server:
```
flask urn
```
